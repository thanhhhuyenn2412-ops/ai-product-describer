# AI Product Description Generator

Ứng dụng web sử dụng AI để tạo mô tả sản phẩm cho ecommerce.

## Công nghệ sử dụng
- HTML
- CSS
- JavaScript
- Gemini API
- Docker

## Cách chạy bằng Docker

Build image:

docker build -t ai-product .

Run container:

docker run -p 8080:80 ai-product

Sau đó mở trình duyệt:

http://localhost:8080
